# Restaurant Recommendation Project

## Description
This project demonstrates a CI/CD pipeline using GitHub, Jenkins, and Docker.
Any change pushed to GitHub is automatically built and deployed.

## Technologies Used
- Node.js
- MySQL
- Docker
- Docker Compose
- Jenkins
- GitHub
- SSH
- ngrok

## CI/CD Workflow
1. Developer updates the code.
2. Code is pushed to GitHub using a bash script.
3. Jenkins automatically pulls the latest code.
4. Jenkins builds a Docker image.
5. Docker Compose deploys the application container.
6. The application is updated automatically.

## Docker Setup
- The application runs inside a single Docker container.
- Docker Compose is used to manage container deployment.

## Jenkins
- Jenkins runs as a system service.
- Jenkinsfile defines the CI/CD pipeline.
- Any push to GitHub triggers automatic build and deployment.
